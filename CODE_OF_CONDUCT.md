# Código De Conducta

## Compromiso

Este proyecto busca mantener un espacio práctico, respetuoso y útil para colaborar.

## Comportamiento Esperado

- Respeta a las personas y al tiempo de revisión.
- Expón problemas técnicos con claridad.
- Critica el cambio, no a la persona.
- Prioriza evidencia, reproducción y contexto.

## Comportamiento No Aceptable

- ataques personales
- tono abusivo o hostil
- spam o ruido deliberado
- publicación irresponsable de vulnerabilidades explotables

## Aplicación

Los mantenedores pueden moderar conversaciones, rechazar contribuciones o cerrar interacciones que rompan estas normas.
